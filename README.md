# Gouach controller

## Peripherals pinout
### Throttle
- +5V
- ADC_THROTTLE
- GND

### Torque sensor
- +5V
- PFS
- ADC_TORQUE
- CRUISE
- GND

### Brake
- +5V
- IO_BRAKE
- ADC_BRAKE
- GND

### Motor
3 Phases
- +5V
- HALL_1
- HALL_2
- HALL_3
- TEMPERATURE
- GND

### UART
- +5V
- Tx
- Rx
- GND

### UART
- CAN_H
- CAN_L

### USB
- +5V
- D+
- D-
- GND
